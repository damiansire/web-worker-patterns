🌐 Angular + Shared Worker

Ejemplo 05: Comunicación entre múltiples pestañas con Angular

💡 ¿Qué demuestra este ejemplo? Un Shared Worker puede ser compartido por múltiples pestañas. Con Angular, encapsulamos esta lógica en un servicio que gestiona la comunicación entre pestañas usando RxJS.
📖 Ver Código - ¿Cómo funciona?

Servicio de Shared Worker

import { injectable } from '@angular/core';
import { Subject } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class SharedWorkerService {
    private sharedWorker: SharedWorker;
    private port: MessagePort;
    private messages = new Subject<string>();
    private tabId: string;
    
    constructor() {
        this.tabId = this.generateTabId();
        this.setupSharedWorker();
    }
    
    private setupSharedWorker(): void {
        this.sharedWorker = new SharedWorker('./shared_worker.js');
        this.port = this.sharedWorker.port;
        
        // Iniciar conexión
        this.port.start();
        
        // Escuchar mensajes de todas las pestañas
        this.port.onmessage = (e) => {
            console.log('Mensaje recibido en pestaña', this.tabId, ':', e.data);
            this.messages.next(e.data.message);
        };
    }
    
    sendMessage(message: string): void {
        this.port.postMessage({
            type: 'broadcast',
            tabId: this.tabId,
            message
        });
    }
    
    getMessages() {
        return this.messages.asObservable();
    }
    
    private generateTabId(): string {
        return `tab-${Date.now()}-${Math.random()}`;
    }
}
⚠️ Nota de compatibilidad: Los Shared Workers no son soportados en todos los navegadores. Safari no los soporta. Firefox y Chrome sí.
