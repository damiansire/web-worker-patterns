🚀 Comunicación Básica con Web Workers

Ejemplo 01: Enviando y recibiendo mensajes

💡 ¿Qué hace este ejemplo? Este es el "Hola Mundo" de los Web Workers. Escribe un mensaje y envíalo al worker. El worker lo recibirá, lo procesará y te responderá de vuelta.
📖 Ver Código - ¿Cómo funciona?

1️⃣ Crear el Worker (main.js)

// Crear una nueva instancia del worker
const myWorker = new Worker('worker.js');

2️⃣ Enviar mensaje al Worker (main.js)

// Enviar datos al worker usando postMessage
myWorker.postMessage('¡Hola Worker!');

3️⃣ Recibir mensaje en el Worker (worker.js)

// Escuchar mensajes del hilo principal
self.onmessage = function(e) {
    const mensaje = e.data;
    console.log('Worker recibió:', mensaje);
    
    // Procesar y responder
    const respuesta = `Procesé tu mensaje: "${mensaje}"`;
    self.postMessage(respuesta);
};

4️⃣ Recibir respuesta del Worker (main.js)

// Escuchar respuestas del worker
myWorker.onmessage = function(e) {
    const respuesta = e.data;
    console.log('Hilo principal recibió:', respuesta);
};
