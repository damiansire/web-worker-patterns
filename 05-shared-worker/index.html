🌐 Shared Worker

Ejemplo 05: Comunicación entre múltiples pestañas

💡 ¿Qué demuestra este ejemplo? Un Shared Worker puede ser compartido por múltiples contextos de navegación (pestañas, iframes, ventanas). Es ideal para compartir estado, gestionar conexiones WebSocket compartidas, o sincronizar datos entre pestañas.
⚠️ Nota de compatibilidad: Los Shared Workers no son soportados en todos los navegadores. Safari no los soporta. Firefox y Chrome sí. Ver compatibilidad
📖 Ver Código - ¿Cómo funciona?

1️⃣ Crear Shared Worker (main.js)

// Crear el Shared Worker (compartido entre pestañas)
const sharedWorker = new SharedWorker('shared_worker.js');

// Obtener el puerto de comunicación
const port = sharedWorker.port;

2️⃣ Conectar y Enviar (main.js)

// Iniciar la conexión
port.start();

// Enviar mensaje al Shared Worker
port.postMessage({
    type: 'message',
    tabId: 'tab-123',
    message: 'Hola desde pestaña 1'
});

3️⃣ Manejar Conexiones (shared_worker.js)

const connections = [];

// Evento cuando una pestaña se conecta
self.onconnect = function(e) {
    const port = e.ports[0];
    connections.push(port);
    
    // Escuchar mensajes de esta pestaña
    port.onmessage = function(event) {
        const { type, message } = event.data;
        
        // Reenviar a TODAS las pestañas
        connections.forEach(conn => {
            conn.postMessage(message);
        });
    };
    
    port.start();
};

4️⃣ Recibir Mensajes (main.js)

// Escuchar mensajes del Shared Worker
port.onmessage = function(e) {
    console.log('Mensaje recibido:', e.data);
    // Este mensaje puede venir de cualquier pestaña
};
