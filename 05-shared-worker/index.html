🌐 Shared Worker

Ejemplo 05: Comunicación entre múltiples pestañas

💡 ¿Qué demuestra este ejemplo? Un Shared Worker puede ser compartido por múltiples contextos de navegación (pestañas, iframes, ventanas). Es ideal para compartir estado, gestionar conexiones WebSocket compartidas, o sincronizar datos entre pestañas.
⚠️ Nota de compatibilidad: Los Shared Workers no son soportados en todos los navegadores. Safari no los soporta. Firefox y Chrome sí. Ver compatibilidad
